# alphastorm-ai-trading

AI-driven algorithmic trading framework with MT5 integration, walk-forward testing, and real-time execution pipeline.

## Quick start

### Clone

If you have GitHub CLI installed:

```bash
gh repo clone jun1-0/alphastorm-ai-trading
```

If `gh` is not installed, use git directly:

```bash
git clone https://github.com/jun1-0/alphastorm-ai-trading.git
```

### Open in VS Code

```bash
code alphastorm-ai-trading
```
